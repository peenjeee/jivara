# Jivara Backend

Backend Jivara dibangun dengan **Express.js**, **TypeScript**, dan **Node.js**.

## Menjalankan lokal

```bash
npm install
npm run dev
```

## Script

```bash
npm run dev    # Menjalankan server development dengan hot-reload
npm run build  # Mengkompilasi TypeScript ke JavaScript (dist/)
npm start      # Menjalankan hasil build produksi
```
